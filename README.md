# Telecom Churn Analysis Dashboard

## Overview
This project focuses on analyzing customer churn in the telecom industry using Power BI.  
The dashboard provides insights into customer behavior, churn trends, contract types, payment methods, and services that influence customer retention.

The goal of this project is to help businesses identify high-risk customers and improve customer retention strategies through data-driven insights.

---

# Tools & Technologies Used
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Visualization

---

# Dataset Information
The dataset contains telecom customer information including:
- Customer demographics
- Subscription services
- Contract details
- Monthly charges
- Total charges
- Churn status

---

# Key KPIs
- Total Customers
- Churn Customers
- Churn Rate
- Monthly Revenue
- Average Monthly Charges

---

# Dashboard Insights
The dashboard helps identify:
- Customers most likely to churn
- Churn trends based on contract type
- Impact of payment methods on churn
- Relationship between tenure and churn
- Service usage patterns of churned customers

---

# Features
- Interactive dashboard
- Dynamic filters
- KPI cards
- Customer segmentation
- Churn trend analysis
- Service-wise churn analysis
